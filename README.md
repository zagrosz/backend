# zaGrosz backend

## Running

In order to run the service you will need the following dependencies

- JDK v1.8
- Maven v3

## How to run

### Run from terminal
`mvn spring-boot:run`

### Run from Intellij

- Right click on `/src/main/java/pl/zagrosz/ZaGroszApplication.java`
- Select "Run ZaGroszApplication"
